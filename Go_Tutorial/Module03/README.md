# Module 3 – Go with the Flow: Conditionals & Loops

## OBJECTIVE

Learn how to use conditionals and loops to control the flow of Go programs.

## LEARNING OUTCOMES

- Use if, else if, and else for decision-making.
- Use switch for cleaner multi-condition handling.
- Implement looping with for and range.
- Combine conditionals and loops in small programs.

## ASSIGNMENT

- Create a folder named `module3_assignment`.
- Write a program that:
  - Checks if a user’s age is eligible to vote using if-else.
  - Uses a switch statement to print a message based on the current day.
  - Prints numbers from 1 to 10 using a for loop.
  - Iterates over an array of your top 3 favorite movies using a range loop.
- Save, run, and screenshot your output.
